== Description ==

Alley Business Tootkit help you to create post types that needed for any business. This toolkit will help you create Testimonial, Promotions, Teams and many more.

The toolkit can also be use to import demo content for the post types in the website.


== Installation ==

1. Upload `alley-business-toolkit.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Can I use this plugin for any themes? =

Yes, you can use this plugin to show post type in any themes you want.
